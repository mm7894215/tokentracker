# Milestones

## M1 - Requirements & Acceptance

- Entry criteria: Canvas refreshed; scope confirmed.
- Exit criteria: Requirement analysis + acceptance criteria drafted and reviewed.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal

- Entry criteria: M1 complete.
- Exit criteria: Proposal + tasks + spec deltas drafted.
- Required artifacts: `openspec/changes/<id>/proposal.md`, `tasks.md`, `specs/**/spec.md`.

## M3 - Test Strategy

- Entry criteria: M2 complete.
- Exit criteria: Test strategy + milestone plan approved.
- Required artifacts: `test-strategy.md`, `milestones.md`.

## M4 - Regression & Integration

- Entry criteria: Guardrails implemented.
- Exit criteria: Regression tests pass; coverage gaps documented.
- Required artifacts: Test logs + verification report.

## M5 - Release & Monitoring

- Entry criteria: M4 complete.
- Exit criteria: Release checklist satisfied; monitoring notes recorded.
- Required artifacts: `verification-report.md`.
