# Milestones

## M1 - Requirements & Acceptance

- Entry criteria: 场景选择策略确认（N=3, pin/exclude）。
- Exit criteria: 需求与验收标准完成。
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal

- Entry criteria: M1 完成。
- Exit criteria: 提案、任务清单、规格增量完成。
- Required artifacts: `proposal.md`, `tasks.md`, `specs/interaction-sequence-canvas-generator/spec.md`.

## M3 - Implementation

- Entry criteria: 提案获批。
- Exit criteria: 生成器、配置、测试与文档完成。
- Required artifacts: 脚本、测试、`interaction_sequence.canvas` 示例输出。

## M4 - Verification

- Entry criteria: M3 完成。
- Exit criteria: 相关测试通过，回归声明记录。
- Required artifacts: `verification-report.md`.
