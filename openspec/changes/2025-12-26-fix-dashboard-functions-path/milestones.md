# Milestones

## M1 - Requirements & Acceptance

- Entry criteria: Issue reproduced and scope agreed.
- Exit criteria: `requirements-analysis.md` and `acceptance-criteria.md` complete.
- Required artifacts: Requirement analysis, acceptance criteria.

## M2 - OpenSpec Proposal (if applicable)

- Entry criteria: M1 complete.
- Exit criteria: `proposal.md`, `tasks.md`, and spec delta drafted.
- Required artifacts: Proposal, tasks, spec delta.

## M3 - Unit Test Coverage

- Entry criteria: Proposal approved.
- Exit criteria: Resolver unit tests fail before fix and pass after fix.
- Required artifacts: Test file + test output.

## M4 - Regression & Integration

- Entry criteria: M3 complete.
- Exit criteria: Manual curl verification and dashboard refresh validated.
- Required artifacts: Verification notes with status codes.

## M5 - Release & Monitoring

- Entry criteria: M4 complete.
- Exit criteria: Docs updated; deployment note recorded if applicable.
- Required artifacts: Updated docs and verification report.
